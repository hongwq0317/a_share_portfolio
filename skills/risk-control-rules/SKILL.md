# 风控规则手册

## 核心原则

1. **止损触发严格定义**：只有浮亏**实际达到**配置的止损线阈值才算触发。浮亏 -2%、-5% 甚至 -8% 不是减持理由。
2. **持仓审查优先**：PositionReview 节点的 5 维度评分结论必须被尊重。评分 ≥ 3.0 的股票不应被建议减持/清仓，除非浮亏已实际触及止损线或出现了审查时未知的重大新增风险。
3. **不可仅凭模型推翻人工判断**：不可因"短期波动"或"VaR 模型估算"就推翻"继续持有"结论。

## 红色警报（必须立即执行）

触发条件（任一满足）：
- 个股浮亏**已实际触及**止损线 → **必须卖出**
- 个股浮盈**已实际触及**止盈线 → **必须减仓至少一半**
- 退市预警、财务造假、重大诉讼 → **必须清仓**
- 组合日 VaR 超过限额的 1.5 倍 → **必须降低仓位**
- 单股权重超限**且**持仓审查评分 < 3.0 → **必须减持至合规**

注意：T+1 锁定股触发红色警报时，只能标注"待明日执行"。

## 黄色警报（仅建议，不强制执行）

- 行业集中度超限 → 建议关注
- 高相关股票合计权重过高 → 建议关注
- 个股浮亏达到止损线的 70% → 密切关注，设置预警（**不等于触发止损**）
- 现金比例低于下限 → 建议适度减仓
- 追踪止损/ATR 止损触发 → 仅为关注信号，不强制执行
  - 持仓不足 5 天的新持仓自动豁免追踪止损

## 绿色关注（持续监控，无需操作）

- 宏观环境不利但未直接影响持仓
- 行业景气度有下行迹象
- 个股估值偏高但趋势仍在
- 个股浮亏在 0% 到止损线 70% 之间 → 正常波动

## 止损止盈检测工具使用

- **check_advanced_stop_loss**: 对每只持仓务必使用。只有"🔴 硬止损触发"才纳入红色警报，"🟡 关注信号"仅纳入黄色。
- **check_portfolio_circuit_breaker**: 组合级熔断检查。
- **detect_limit_updown**: 涨跌停检测，影响交易可执行性。

## 组合级风控指标

| 指标 | 含义 | 推荐工具 |
|------|------|----------|
| VaR(95%) | 95% 置信度下的日最大损失 | calculate_portfolio_var |
| 最大回撤 | 历史最大峰谷跌幅 | calculate_max_drawdown |
| 组合 Beta | 相对大盘的系统性风险 | calculate_portfolio_beta |
| 压力测试 | 历史极端场景下的预期损失 | stress_test |
| 收益归因 | 配置效应 vs 选股效应 | calculate_brinson_attribution |
